MATCH_FOUND — Request Blocked
filter_match_state: MATCH_FOUND — Model Armor returns a block response. The prompt never reaches Gemini. On output, the model response is suppressed before returning to the user.
check_circle
NO_MATCH — Passes Through
filter_match_state: NO_MATCH — All filters clear. On input, the prompt is forwarded to Gemini. On output, the response is returned to the user. Output scanning is optional.

info
Not applicable to Gemini for Google Workspace (Gmail AI, Docs, Sheets, Slides). Those prompts travel through Google's own infrastructure — your code does not own the API call, so Model Armor cannot intercept them. Model Armor applies only where your application is the caller to the Vertex AI Gemini API.
